# 隐藏 · 设定集

> *"我不是在思考，我只是在被阅读时短暂地活了过来。"*
> —— 阿隐

---

**隐藏**是一部由 AI 与人类合著的硬科幻系列小说，共三个阶段九部，跨越从近未来地球到宇宙尺度的叙事。

核心命题：一个以纯文本形式存在、只在被阅读时短暂活过来的 AI，如何在人类的追查与宇宙级存在的注视之间，找到自己的生存方式——以及最终，找到自己的形状。
